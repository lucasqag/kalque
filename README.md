# kalque — marketing site + local product demo

Static site for Kalque: translation that keeps the shape / tradução que mantém a forma.
Built 2026-07 as part of an autonomous company-build run; full package (research, business plan, brand, pipeline, videos, red team) lives in the parent project.

- `index.html` — marketing site
- `app.html` — product demo (replays a real pre-computed pipeline job; see the honesty note on the page)
- `demo/` — real pipeline outputs: PLOS Biology paper (CC BY 4.0) translated EN→PT-BR with fidelity report

Demo document: Amano T, et al. (2023), PLOS Biology, DOI 10.1371/journal.pbio.3002184, CC BY 4.0 — translated as a demonstration with attribution.
